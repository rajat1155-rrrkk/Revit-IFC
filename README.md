# IFC Material Demo

Small standalone showcase app for the IFC material availability concept.

## What it shows

- IFC model report summary
- available materials
- partially available materials
- unavailable materials
- covered cost and shortage value

The demo reads a generated JSON report and renders it as a browser-friendly dashboard.

## Run locally

```bash
python3 scripts/serve.py
```

Then open `http://localhost:4173`.

## Sample data

The bundled sample report lives at `public/data/report.json`.

You can also load a different report JSON from the page itself using the upload control.

## Where the report comes from

The source report was generated by the IFC material mapper in the parent workspace:

```bash
python3 tools/material_mapper.py \
  --ifc "IFC 4.3.2.0 (IFC4X3_ADD2)/PCERT-Sample-Scene/Building-Architecture.ifc" \
  --inventory data/material_inventory.csv \
  --json-out output/building-architecture-report.json
```
